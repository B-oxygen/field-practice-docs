# 외부근무사유서 (External Work Reason)

시스템(aLog)에 기록되지 않은 외부 근무시간을 주차별로 사유 제출하는 양식.

## 양식 구조 (HWP, 주차별 1장)
센터 메모: "일자가 많으니 주차별로 작성 / 해당 주차의 마지막 일자로 작성 / 서명필수."
표 1개, 필드 순서:
1. **창업동아리(창업기업)명**
2. **근무일시 / 시간** — 일자별 `YYYY년 MM월 DD일(요일) / HH:MM~HH:MM(N분)` 여러 줄
3. **근무소재지** — 시스템 미등록 장소(실제 장소 기입)
4. **근무 내용** — `[제목]` + 상세
5. **사유** — 외부활동 정당화 + "사실이 아닐 경우 불이익 감수" 확인
하단 날짜 = **해당 주차의 마지막 날(일요일)**, 서명 2인(본인 / 센터 담당자), 수신 = 창업교육센터장.

## 데이터 규칙
- 대상/분(分)은 **제출리스트가 정본**. 주차별로 묶고, 일자별 분의 합 = 제출리스트 합과 일치해야 한다.
- **일자별 분**:
  - 시스템 미등록일 → 제출 근무시간 전부.
  - 부분 등록일 → 부족분만(제출 − 시스템). 제출리스트의 "X분에 대한 근무 사유서 제출" 값.
- **시간대**: 분은 정본, 시각(HH:MM)은 09:00 시작 등으로 산출 후 실제값으로 검토. 괄호 안 (N분)만 유지하면 시각은 조정 가능.
- **작성 날짜**: 주차의 마지막 일자 ≠ 마지막 근무일. 주(월~일)의 **일요일**로 통일(이번 사례의 실수 포인트).

## 절차
1. 제출리스트에서 주차별 (일자, 분, 미등록/부분, 부족분) 추출 → 검산(합계).
2. 주차별 fill sheet 작성(필드 순서대로 복붙용 markdown). 근무내용/사유는 주차 맥락으로 작성.
3. HOP/문서 편집기에서 양식을 주차 수만큼 복제해 붙여넣기 → 날짜(일요일)·서명.
4. 합계가 제출리스트와 일치하는지 최종 대조.

## 주의
- HWP 바이너리는 자동 채우기 비권장 → fill sheet를 만들어 HOP/문서 편집기에서 사람이 작성.
- 근무소재지·실제 시각은 사실이어야 함(허위 시 불인정). 제안값은 반드시 검토·교체.
- 근무내용도 보고서와 같은 AI 문체가 나오기 쉬움 → `references/deslop.md` 참고.
